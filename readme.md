Random Password Generator (Python)

Project Description
This project is an advanced Random Password Generator developed in Python with a graphical user interface. It allows users to generate secure and customizable passwords based on selected criteria such as length, character types, and complexity.

The application enhances security by generating strong passwords and provides a user-friendly interface for better usability. It also includes clipboard functionality to quickly copy generated passwords.

Features

Generate random secure passwords
User-defined password length
Option to include:
Uppercase letters
Lowercase letters
Numbers
Special characters
Strong password generation based on security rules
Easy-to-use graphical interface
Copy password to clipboard functionality
Input validation for better user experience

Tools and Technologies Used

Python
Tkinter (GUI development)
ttk (for styled widgets)
random (for password generation)
string (character sets)
pyperclip (clipboard functionality)
VS Code

How It Works

User enters desired password length
Selects character types (letters, numbers, symbols)
Clicks "Generate Password"
System generates a random secure password
User can copy the password using the "Copy" button

Sample Output

Password Length: 10
Selected Options: Uppercase, Lowercase, Numbers, Symbols

Generated Password: A@7kL9#pQ2
